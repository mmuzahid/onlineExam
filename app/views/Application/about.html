Today, Information Technological advancement has minimized the whole world. &{'application.name'} is an application that allows conducting examination through web. Goal of this project is to provide quick, immediate and easy way to appear the exam and reducing usage of pen and paper. This application allow us to arrange a MCQ (multiple choice question) examination thorough Internet. By signing in the system we can create our own exams, can search existing exams and can participate to those exams. This application supports both single and multiple answered questions. Exam author who created that exam can also configure some important features such as question set is random or fixed, exam validation period in which exam will be activated etc. We can search exam by some keyword like author name, exam title, and exam code. After searching we can start an exam by clicking start button. Then System provides participants an interface almost same as real world MCQ answer paper with questions. Participants fill the answer paper up and submit it to the server by clicking submit test button. After receiving submitted answers system calculates the result and responses result to the participant with correct answer. Participates can view their attended exam results and also able to ask any inquiry about the question. Exam author also can view a summarized report to observe the result status of the exam. Another user of this system is Administrator who has the authorization to manipulate user information and also some exam information.
